# JanSarthi
JanSarthi is an AI-powered platform that helps citizens discover government schemes they are eligible for based on their personal and socio-economic information. The platform simplifies access to government benefits and improves digital inclusion.
🚀 Features
AI-based scheme recommendations
Personalized eligibility checker
Eligibility probability score
Required documents checklist
Direct application links
Location-based scheme filtering
Responsive and user-friendly UI
🧠 How It Works
User enters personal details (age, income, occupation, state, category)
AI eligibility engine evaluates user data
System recommends relevant government schemes
User views eligibility, documents, and apply links
🛠️ Tech Stack

Frontend: React / Next.js
Backend: Node.js / Python
Database: JSON / SQLite / MongoDB
AI Engine: Rule-based eligibility system
