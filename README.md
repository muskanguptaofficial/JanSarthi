🇮🇳 Jansarthi

Jansarthi is an AI-powered platform that helps citizens discover government schemes they are eligible for based on their personal and socio-economic information. The system simplifies access to government benefits and improves awareness.

🚀 Features
AI-based scheme recommendations
Personalized eligibility checker
Eligibility probability score
Required documents checklist
Direct application links
Location-based filtering
Responsive UI
🧠 How It Works
User enters personal details
System evaluates eligibility
Platform recommends suitable schemes
User views documents and application links
🛠️ Tech Stack
Frontend: React / Next.js
Backend: Node.js / Python
Database: JSON / SQLite / MongoDB
AI Engine: Rule-based eligibility system
